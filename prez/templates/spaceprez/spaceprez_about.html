About SpacePrez

SpacePrez is a read-only web delivery system - web pages and API - for GeoSPARQL spatial data. It complies with Content Negotiation by Profile.

Introduction

This is an instance of the new version of OGC LD API - SpacePrez. It is a part of a Linked Data presentation API called Prez. This instance is the first demo version of Prez.
